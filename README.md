# F1-Data-Analysis
Exam project for BigData course at RAF
## Requires
- Python
- Pandas
- PySpark
- Docker
- Databricks Community Edition
- Apache Kafka Docker Image
- Apache Kafka Producer Client
- Apache Kafka Consumer CLinet
- Unofficial F1 [API](https://openf1.org/)
